CSS Design

ตกแต่งเว็บให้สวยด้วย CSS: จากโล่งๆ เป็นพรีเมียม

เมื่อเรามีโครงสร้างบ้านจาก HTML แล้ว ขั้นต่อมาคือการ "ทาสี" และ "จัดเฟอร์นิเจอร์" ซึ่งนั่นคือบทบาทของ CSS (Cascading Style Sheets) ครับ

1. การใช้งาน CSS เบื้องต้น

เราสามารถเขียน CSS ได้ 3 วิธี แต่ที่นิยมที่สุดคือการแยกไฟล์ style.css ออกมาต่างหาก เพื่อความเรียบร้อยของโค้ด

body {
  background-color: #080b14;
  color: white;
  font-family: 'Sarabun', sans-serif;
}

2. สีและฟอนต์ (Colors & Fonts)

การเลือกใช้พาเลทสีที่เข้ากัน จะช่วยให้เว็บไซต์ดูมีความเป็นมืออาชีพมากขึ้น แนะนำให้ใช้สีโทนเข้ม (Dark Mode) ตัดกับสีสะท้อนแสง (Neon/Accent) เพื่อความโดดเด่น

3. Flexbox และ Grid

เป็นเทคนิคการจัดวาง layout ที่ทรงพลังที่สุดในปัจจุบัน ช่วยให้คุณจัดหน้าเว็บให้รองรับมือถือ (Responsive) ได้อย่างง่ายดาย
