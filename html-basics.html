HTML Beginner

พื้นฐาน HTML สำหรับมือใหม่: เริ่มต้นสร้างเว็บไซต์จากศูนย์

หากคุณเปรียบการสร้างเว็บไซต์เหมือนกับการสร้างบ้าน HTML (HyperText Markup Language) ก็คือ "โครงสร้างบ้าน" นั่นเองครับ ไม่ว่าจะเป็นเสาเข็ม ผนัง หรือหลังคา หากไม่มีโครงสร้างเหล่านี้ เราก็ไม่สามารถตกแต่งบ้านให้สวยงามได้

1. HTML คืออะไร?

HTML ไม่ใช่ภาษาโปรแกรมมิ่ง (Programming Language) แต่เป็นภาษา "Markup" ที่ใช้บอกเบราว์เซอร์ว่าส่วนไหนคือหัวข้อ (Heading) ส่วนไหนคือย่อหน้า (Paragraph) หรือส่วนไหนคือรูปภาพ

2. โครงสร้างพื้นฐานของไฟล์ HTML

ทุกไฟล์ HTML จะเริ่มต้นด้วยโครงสร้างหน้าตาแบบนี้เสมอ:

<!DOCTYPE html>
<html>
<head>
  <title>ชื่อเว็บของคุณ</title>
</head>
<body>
  <h1>หัวข้อหลัก</h1>
  <p>เนื้อหาในเว็บ...</p>
</body>
</html>

3. Tag ที่ใช้บ่อยที่สุด

  • <h1> ถึง <h6>: ใช้สำหรับหัวข้อ (h1 ใหญ่ที่สุด)
  • <p>: ใช้สำหรับเนื้อหาที่เป็นข้อความทั่วไป
  • <a>: ใช้สำหรับสร้างลิงก์เชื่อมโยง
  • <img>: สำหรับใส่รูปภาพ
ทิป: การเขียน HTML ที่ดีควรเลือกใช้ Tag ให้เหมาะสมกับเนื้อหา (Semantic HTML) เพื่อช่วยให้ Google เข้าใจเว็บของคุณได้ดีขึ้นครับ
