        📋 ขั้นตอน
        
Step-by-Step Guide

        
ทำตามได้เลย ทีละขั้นตอน

      


      
      

        

          
Step 1

          
สร้างโฟลเดอร์และไฟล์ HTML

        

        

          
เปิด VS Code แล้วสร้างโฟลเดอร์โปรเจกต์ใหม่ จากนั้นสร้างไฟล์ index.html แล้วพิมพ์โค้ดพื้นฐานนี้ (นี่คือโค้ดตัวอย่างสำหรับเริ่มต้นเรียนรู้):

          

            

              📄 index.html
              
            

            
<!DOCTYPE html>
<html lang="th">
<head>
  <meta charset="UTF-8">
  <title>เว็บไซต์แรกของฉัน</title>
</head>
<body>
  <h1>สวัสดีโลก! 🌏</h1>
  <p>นี่คือเว็บไซต์แรกของฉัน</p>
</body>
</html>

          

          

            💡 Pro Tip: กด ! แล้ว Tab ใน VS Code จะสร้าง HTML Boilerplate ให้อัตโนมัติ!
          

        

      


      
      

        

          
Step 2

          
อัปโหลดไฟล์ขึ้น GitHub

        

        

          
    
            
  1. ไปที่ github.com/new แล้วสมัครบัญชีฟรี (ถ้ายังไม่มี)
    2. 
            
  2. กด New repository แล้วตั้งชื่อ (เช่น my-first-website)
    3. 
            
  3. เลือก Public แล้วกด Create repository
    4. 
            
  4. เข้าไปใน Repository ที่สร้าง แล้วกด uploading an existing file
    5. 
            
  5. ลาก-วาง (Drag & Drop) ไฟล์ทั้งหมดจากโฟลเดอร์โปรเจกต์ลงในหน้านั้น
    6. 
            
  6. กด Commit changes เพื่อบันทึก — เสร็จแล้ว! 🎉
    7. 
          

          

            💡 Pro Tip: ไฟล์หลักที่ต้องอัปโหลดคือ index.html และไฟล์ CSS/JS ถ้ามี — GitHub จะเก็บทุกอย่างไว้ให้บนคลาวด์
          

        

      


      
      

        

          
Step 3

          
นำเว็บขึ้นออนไลน์ด้วย Vercel

        

        

          
    
            
  1. ไปที่ vercel.com แล้วกด Sign Up with GitHub
    2. 
            
  2. กด Add New → Project แล้วเลือก Repository ที่สร้างใน GitHub
    3. 
            
  3. กด Deploy รอประมาณ 30 วินาที
    4. 
            
  4. ได้รับ URL สาธารณะ เช่น my-first-website.vercel.app ทันที!
    5. 
          

          

            🚀 ข่าวดี: จากนี้ไปทุกครั้งที่คุณอัปโหลดไฟล์ใหม่บน GitHub เว็บจะอัปเดตเองอัตโนมัติ ไม่ต้องทำอะไรเพิ่ม!
          

        

      

    

  


  
  

    

      

        🚀 Deploy
        
นำเว็บขึ้นออนไลน์ด้วย Vercel

        
ขั้นตอนสุดท้าย — เว็บของคุณจะ Live ภายใน 2 นาที

      

      

        

          
1️⃣

          
สมัคร Vercel

          
ไปที่ vercel.com แล้วกด Sign Up with GitHub

        

        
→

        

          
2️⃣

          
Import Project

          
กด Add New → Project แล้วเลือก Repository ที่สร้างไว้

        

        
→

        

          
3️⃣

          
Deploy!

          
กด Deploy รอ 30 วินาที แล้วได้ URL สาธารณะทันที!

        

      


      

        
🔄 อัปเดตเว็บไซต์ง่ายมาก!

        
เมื่อแก้ไขไฟล์เสร็จแล้ว แค่ไปที่ GitHub Repository ของคุณ กด Add file → Upload files แล้วลาก-วางไฟล์ที่แก้ไขแล้วลงไป จากนั้นกด Commit changes — Vercel จะ Deploy เว็บใหม่ให้อัตโนมัติทันที ไม่ต้องทำอะไรเพิ่มเลย!

        

          ✨ ง่ายแค่นี้เอง: แก้ไฟล์ใน VS Code → อัปโหลดขึ้น GitHub → เว็บอัปเดตเอง
